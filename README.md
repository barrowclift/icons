Icons
===========

###General Information

This is a collection of custom icons I made based off some of the popular
templates online. Feel free to download and tweak them yourselves! (Just be
sure to give proper credit to the template creators).

[Yosemite Folder Template by Coloradan](http://coloradan.deviantart.com/art/Yosemite-Custom-Folder-459481587)<br />
[Yosemite App Icon Template by Nick Keppol](http://martiancraft.com/blog/2014/07/inspecting-yosemite-icons/)
